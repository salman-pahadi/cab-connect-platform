### **For Midjourney:**

**Prompt Format:**
```
/imagine [paste prompt] --v 6 --style raw --ar 16:9
```

**Example:**
```
/imagine Professional minimalist logo for FIJI CAB CONNECT ride-hailing 
app, simple geometric taxi icon in emerald green #10b981, bold sans-serif 
black text, Uber-like clean modern style, white background, horizontal 
layout, vector-style graphic design --v 6 --style raw --ar 16:9
```

**Parameters:**
- `--v 6` = Latest version
- `--style raw` = Less stylized, more literal
- `--ar 16:9` = Aspect ratio for horizontal logo
- `--no gradient, shadows` = Cleaner vector look

**Refinement Commands:**
```
After generation:
- Click "Upscale" on best option
- Use "Vary (Strong)" for different variations
- Use "Zoom Out" if logo is cropped
```
