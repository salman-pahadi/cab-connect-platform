## üìñ HOW TO USE

### **For New Projects:**

1. **Schedule 2-hour brand workshop** with client
2. **Complete ALL 93 fields** during workshop (no blanks)
3. **Get stakeholder sign-off** from all decision makers
4. **Lock the brand kit** - this becomes source of truth
5. **THEN start development** - reference this for every decision

### **During Development:**

**Before ANY design decision, ask:**
1. Does this align with brand personality? (Section 2)
2. Do colors match palette? (Section 3)
3. Does font match typography? (Section 4)
4. Does message match voice? (Section 6)
5. Will target audience understand? (Section 7)
6. Does this differentiate from competitors? (Section 8)
7. Does this comply with restrictions? (Section 10)

**If answer not in brand kit:**
- STOP development
- Consult client
- Update brand kit
- Get approval
- THEN proceed
