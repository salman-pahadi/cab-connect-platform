## üìê **PART 1: SETUP YOUR CANVAS**

### **Step 1: Create New Design**

1. **Go to:** canva.com
2. **Click:** "Create a design" (top right)
3. **Select:** "Custom size"
4. **Enter dimensions:**
   - Width: **3000 px**
   - Height: **1500 px**
   - (This is 2:1 ratio, perfect for horizontal logos)
5. **Click:** "Create new design"

**Why these dimensions?**
- High resolution for printing
- Horizontal format for logo layout
- Easy to scale down for app/web use

---

### **Step 2: Setup Background**

1. **Click** on the canvas background
2. **Color picker** appears on top toolbar
3. **Enter:** `#FFFFFF` (pure white)
4. **Apply**

**Result:** Clean white canvas ready for design

---

## üé® **PART 2: SETUP BRAND COLORS**

### **Step 3: Add Your Brand Colors to Palette**

1. **Click** "Elements" (left sidebar)
2. **Scroll down** to "Shapes"
3. **Click** on a square shape
4. **Click** the color tile (shows current color)
5. **Click** "+" next to "Document colors"
6. **Enter HEX codes** one by one:

**Add these colors:**
```
Primary:   #10B981  (Emerald Green)
Secondary: #0891B2  (Ocean Blue)
Accent:    #F59E0B  (Amber Gold)
Text:      #000000  (Black)
White:     #FFFFFF  (White)
```

7. **Click** "Add new color" for each
8. **Delete** the test square you created

**Result:** Brand colors saved in palette for easy access

---

## üöó **PART 3: CREATE THE TAXI ICON**

### **OPTION A: Simple Geometric Taxi (Recommended)**

#### **Step 4: Create Taxi Body**

1. **Click** "Elements" ‚Üí "Shapes"
2. **Search:** "rectangle"
3. **Drag** a rounded rectangle to canvas
4. **Resize:**
   - Width: ~400px
   - Height: ~250px
5. **Round the corners:**
   - Click shape
   - Drag corner radius slider to ~30
6. **Color:** Apply emerald green `#10B981`
7. **Position:** Left side of canvas

#### **Step 5: Add Taxi Windows (Headlights)**

1. **Add** 2 circles (Elements ‚Üí Shapes ‚Üí Circle)
2. **Size:** ~80px diameter each
3. **Color:** White `#FFFFFF`
4. **Position:** Bottom front of taxi body (like headlights)
5. **Align:** Horizontally spaced evenly

#### **Step 6: Add Roof Light**

1. **Add** small rectangle
2. **Size:** ~60px wide √ó ~40px tall
3. **Color:** Amber gold `#F59E0B`
4. **Round corners** slightly (~10)
5. **Position:** Top center of taxi

#### **Step 7: Group Taxi Icon**

1. **Select all taxi elements** (body + circles + roof light)
   - Click and drag to select all
   - OR hold Shift and click each element
2. **Right-click** ‚Üí "Group"
3. **Resize group** to ~500px wide (maintain proportions)

**Result:** Simple, clean taxi icon ‚úÖ

---

### **OPTION B: Use Canva's Icon Library (Faster)**

#### **Alternative Step 4-7:**

1. **Click** "Elements" (left sidebar)
2. **Search:** "taxi icon"
3. **Filter by:** Graphics
4. **Look for:** Simple, flat, minimal taxi icons
5. **Select** a clean geometric one
6. **Resize** to ~500px wide
7. **Change color** to emerald green `#10B981`
8. **Add roof light:** Small rectangle on top, amber gold `#F59E0B`

**Recommended searches:**
- "taxi front view"
- "cab icon"
- "car simple"
- "vehicle front"

**Pro tip:** Look for icons that are single color and geometric, NOT detailed illustrations.

## üî§ **PART 4: ADD TYPOGRAPHY**

### **Step 8: Add Company Name**

1. **Click** "Text" (left sidebar)
2. **Select** "Add a heading"
3. **Type:** `FIJI CAB CONNECT`
4. **Position:** Right side of taxi icon

#### **Step 9: Format Text**

1. **Select** the text
2. **Font:** Plus Jakarta Sans
   - If not available, use: **Montserrat Bold** or **Poppins Bold**
3. **Weight:** ExtraBold or Bold
4. **Size:** ~180-220px (adjust to fit)
5. **Color:** Black `#000000`
6. **Letter spacing:** +5 to +10 (slightly wider)
7. **Line height:** 1.0 to 1.1 (tight)
8. **Alignment:** Left

**Can't find Plus Jakarta Sans?**
Use these alternatives (in order):
1. **Montserrat ExtraBold** ‚≠ê Best alternative
2. **Poppins Bold**
3. **Inter Bold**
4. **Work Sans Bold**

#### **Step 10: Format Layout**

**Option 1: Single Line (if fits)**
```
[Taxi Icon]  FIJI CAB CONNECT
```

**Option 2: Two Lines (if too long)**
```
[Taxi Icon]  FIJI CAB
             CONNECT
```

**Spacing:**
- Gap between icon and text: ~150-200px
- Line spacing (if two lines): 0px (tight)

---

## üéØ **PART 5: ALIGNMENT & SPACING**

### **Step 11: Align Elements**

1. **Select** taxi icon
2. **Select** text (hold Shift to multi-select)
3. **Top toolbar** ‚Üí "Position"
4. **Click** "Align center vertically" (middle option)

**Result:** Icon and text centered on same baseline

### **Step 12: Position on Canvas**

1. **Select all elements** (icon + text)
2. **Group them** (Right-click ‚Üí Group)
3. **Position:**
   - Horizontally: Slightly left of center
   - Vertically: Perfect center
4. **Leave breathing room** on all sides

**Spacing guide:**
- Minimum 200px from left edge
- Minimum 200px from right edge
- Centered vertically

## ‚ú® **PART 6: CREATE VARIATIONS**

### **Step 13: Create Horizontal Version**

1. **Your current design** is already horizontal ‚úÖ
2. **Duplicate page** (Click "+" at bottom)
3. **Adjust spacing** if needed
4. **Save as:** "Logo-Horizontal"

---

### **Step 14: Create Stacked Version**

1. **Duplicate page** again
2. **Ungroup** icon and text
3. **Rearrange:**
   ```
       [Taxi Icon]
         (center)
     
     FIJI CAB CONNECT
       (center below)
   ```
4. **Space between:** ~100-150px
5. **Center align** both elements
6. **Save as:** "Logo-Stacked"

---

### **Step 15: Create Icon-Only Version (App Icon)**

1. **Duplicate page** again
2. **Delete** text
3. **Keep** only taxi icon
4. **Resize canvas:**
   - Click "Resize" (top toolbar)
   - Change to: 1024px √ó 1024px (square)
5. **Center** taxi icon perfectly
6. **Add padding** around icon (~150px from edges)
7. **Save as:** "Logo-Icon-Only"

**This is your app icon!** üì±

---

### **Step 16: Create White Version (Dark Backgrounds)**

1. **Duplicate horizontal page**
2. **Change background** to black `#000000` or dark blue `#1E293B`
3. **Change taxi icon** to white `#FFFFFF`
4. **Change text** to white `#FFFFFF`
5. **Keep roof light** amber gold `#F59E0B` (pops on dark)
6. **Save as:** "Logo-White-Version"

---

### **Step 17: Create Single Color Version**

1. **Duplicate page**
2. **Change ALL elements** to single color:
   - Taxi icon: Emerald green `#10B981`
   - Roof light: Emerald green `#10B981` (same)
   - Text: Emerald green `#10B981`
3. **Background:** White
4. **Save as:** "Logo-Single-Color"

**Use case:** Embroidery, stamps, simple printing

## üé® **DESIGN TIPS & BEST PRACTICES**

### **DO's:**

‚úÖ **Keep it simple** - Less is more
‚úÖ **Use grid/guides** - Canva has ruler and guides (View ‚Üí Show Rulers)
‚úÖ **Test at small sizes** - Zoom out to see if readable
‚úÖ **Maintain consistent spacing** - Use same gaps throughout
‚úÖ **Align everything** - Use alignment tools constantly
‚úÖ **Save frequently** - Canva autosaves but be safe
‚úÖ **Test on dark backgrounds** - Make sure white version works

### **DON'Ts:**

‚ùå **Don't use gradients** - Keep it flat
‚ùå **Don't use too many colors** - Stick to brand palette
‚ùå **Don't use thin lines** - Won't scale well
‚ùå **Don't use complex details** - Keep it geometric
‚ùå **Don't use decorative fonts** - Stay professional
‚ùå **Don't ignore spacing** - Cramped = unprofessional
‚ùå **Don't use drop shadows** - Keep it flat and modern

## üîß **TROUBLESHOOTING**

### **Problem: Font not available**

**Solution:**
1. Use **Montserrat ExtraBold** instead
2. Or use **Poppins Bold**
3. Both are free on Canva and look very similar

---

### **Problem: Icon looks childish**

**Solution:**
1. Reduce corner rounding
2. Use more angular/geometric shapes
3. Keep proportions simple
4. Remove unnecessary details

---

### **Problem: Logo too wide**

**Solution:**
1. Make icon smaller (350-400px)
2. Reduce letter spacing in text
3. Or switch to stacked layout

---

### **Problem: Text hard to read at small size**

**Solution:**
1. Increase font size/weight
2. Increase letter spacing slightly
3. Use bolder font weight
4. Simplify to single line if possible

---

### **Problem: Colors don't match exactly**

**Solution:**
1. Always use HEX codes, not color picker
2. Type in exact codes: #10B981, #F59E0B, #000000
3. Save to brand palette for consistency

## üí° **PRO TIPS**

### **Tip 1: Use Canva's Alignment Tools**

- **Position button** (top toolbar) is your best friend
- Use "Align center" for everything
- Use "Tidy up" to space elements evenly
- Enable rulers (View ‚Üí Show Rulers)

### **Tip 2: Duplicate Before Experimenting**

- Always duplicate your page before making big changes
- Keep previous versions to compare
- You can always go back

### **Tip 3: Test at Multiple Sizes**

- Zoom out to 25% - does it still look good?
- Check at 10% - is icon recognizable?
- If yes, you have a scalable logo ‚úÖ

### **Tip 4: Get Feedback**

Before finalizing:
- Show it to 3-5 people
- Ask: "What company is this?"
- If they say "taxi/ride service" ‚Üí Success ‚úÖ
- If confused ‚Üí Simplify
