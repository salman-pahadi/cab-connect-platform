# Phase 1 Overview

## Objective
Build a stable, working cab booking system that allows passengers to book rides, drivers to accept and complete rides, and admins to manage drivers.

## Focus
- Reliability
- Correct ride lifecycle
- Clean backend foundation

## Non-Goals
- Online payments
- Ratings
- Scheduling
- iOS app
- Advanced analytics

Phase 1 is not about optimisation. It is about correctness.
