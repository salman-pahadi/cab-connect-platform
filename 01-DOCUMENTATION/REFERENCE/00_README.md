# Cab Connect – Phase 1

This repository contains the Phase 1 (MVP) development of **Cab Connect**, a marketplace cab booking platform operating in Fiji.

## Phase
Phase 1 – MVP

## Platforms
- Android Passenger App
- Android Driver App
- Backend APIs
- Minimal Admin Panel

## Tech Stack
- Backend: FastAPI (Python)
- Database: PostgreSQL
- Hosting: AWS (EC2, RDS, S3)
- Auth: Phone + OTP
- Payments: Cash only

## Documentation
All project decisions and execution rules are documented inside `/docs`.

**Code must follow documentation, not assumptions.**
