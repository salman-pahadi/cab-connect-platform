# Phase 1 Locked Decisions

- Platform: Android (Passenger + Driver)
- Backend: FastAPI
- Database: PostgreSQL
- Hosting: AWS
- Auth: Phone + SMS OTP
- Payments: Cash only
- Ride assignment: Broadcast, first accept wins
- Real-time: REST polling
- Scheduling: Not included
- Ratings: Not included
- Ride completion: Driver marks completed
- Cancellation: Passenger until driver arrives
- Admin: Limited control only
